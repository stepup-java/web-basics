Q1. Which control flow statement is used for conditional execution?

âœ“ Correct Answer: if-else

The if-else statement is used for conditional execution. It evaluates a boolean condition and executes different code blocks based on whether the condition is true or false. It's the most basic decision-making structure in Java.

Q2. Which statement is best for multi-way branching based on a single variable?

âœ“ Correct Answer: switch

The switch statement is ideal for multi-way branching when you need to compare a single variable against multiple constant values. It's more readable and efficient than multiple if-else statements for this purpose.

Q3. Which loop is best when you know the exact number of iterations?

âœ“ Correct Answer: for

The for loop is best when you know the exact number of iterations. It combines initialization, condition, and increment/decrement in one line, making it compact and readable for counter-based loops.

Q4. Which loop checks the condition before executing the loop body?

âœ“ Correct Answer: Both while and for

Both while and for loops check the condition before executing the loop body (entry-controlled loops). If the condition is false initially, the loop body never executes. The do-while loop is an exit-controlled loop.

Q5. Which loop guarantees at least one execution of the loop body?

âœ“ Correct Answer: do-while

The do-while loop guarantees at least one execution because it checks the condition after executing the loop body (exit-controlled). This is useful when you need to execute code at least once, like showing a menu.

Q6. Which keyword is used to exit a loop prematurely?

âœ“ Correct Answer: break

The break keyword immediately exits the loop, transferring control to the statement after the loop. It's commonly used to exit when a specific condition is met or in switch statements to prevent fall-through.

Q7. Which keyword skips the current iteration and moves to the next?

âœ“ Correct Answer: continue

The continue keyword skips the remaining code in the current iteration and jumps to the next iteration. It's useful when you want to skip processing for certain values without exiting the entire loop.

Q8. What is the best way to iterate through all elements of an array?

âœ“ Correct Answer: Enhanced for loop (for-each)

The enhanced for loop (for-each) is the best way to iterate through all elements when you don't need the index. Syntax: for (Type element : array). It's more readable and less error-prone than traditional loops.

Q9. Which keyword in a switch statement handles all unmatched cases?

âœ“ Correct Answer: default

The default keyword in a switch statement handles all cases that don't match any case label. It's optional but recommended for handling unexpected values. The default case can appear anywhere but is typically placed last.

Q10. What prevents fall-through in switch statements?

âœ“ Correct Answer: break

The break statement prevents fall-through in switch statements. Without break, execution continues to the next case (fall-through behavior). While sometimes intentional, it's usually a bug, so always use break unless you specifically want fall-through.

Q11. Which data types can be used in a switch statement?

âœ“ Correct Answer: int, char, String, enum

Switch statements support: byte, short, char, int (and their wrapper classes), String (since Java 7), and enum types. They do NOT support long, float, double, or boolean. Use if-else for these types.

Q12. What happens if the condition in a while loop is always true?

âœ“ Correct Answer: Infinite loop

If the condition is always true, the loop runs infinitely (infinite loop). This is sometimes intentional (like while(true) with break inside), but usually it's a bug. The program will hang unless you use break or return to exit.

Q13. How many times will this loop execute? for(int i = 5; i < 5; i++) { }

âœ“ Correct Answer: 0

The loop executes 0 times because the condition (i < 5) is false from the start (i is initialized to 5). For loops check the condition before the first iteration, so if it's false initially, the body never executes.

Q14. How many times will this loop execute? for(int i = 0; i < 5; i++) { }

âœ“ Correct Answer: 5

The loop executes 5 times (i = 0, 1, 2, 3, 4). When i becomes 5, the condition (i < 5) is false, so the loop stops. Remember: the condition is checked before each iteration, including the first.

Q15. What are loops inside loops called?

âœ“ Correct Answer: Nested loops

Loops inside loops are called nested loops. The inner loop completes all its iterations for each iteration of the outer loop. They're commonly used for 2D arrays, matrices, and generating patterns.

Q16. In nested loops, which loop does a break statement exit?

âœ“ Correct Answer: The innermost loop

A break statement exits only the innermost loop containing it. To break out of outer loops, you can use labeled break statements. Example: outerLoop: for(...) { for(...) { break outerLoop; } }

Q17. How can you break out of multiple nested loops at once?

âœ“ Correct Answer: Both labeled break and return

You can use either a labeled break (break labelName;) to exit specific outer loops, or a return statement to exit the entire method. Labeled break is cleaner when you only want to exit loops, not the method.

Q18. What is used to check multiple conditions in sequence?

âœ“ Correct Answer: else if

The else if clause checks multiple conditions in sequence. Once a condition is true, its block executes and the rest are skipped. This is more efficient than separate if statements, which all get evaluated.

Q19. What is a shorthand for simple if-else statements?

âœ“ Correct Answer: Ternary operator (?:)

The ternary operator (condition ? valueIfTrue : valueIfFalse) is a shorthand for simple if-else statements. Example: int max = (a > b) ? a : b; It's concise but should only be used for simple cases to maintain readability.

Q20. How many times does a do-while loop execute if the condition is false initially?

âœ“ Correct Answer: 1

A do-while loop executes at least once even if the condition is false initially, because it checks the condition after the first execution. This is the key difference from while and for loops.

Q21. What happens if you forget the semicolon after a do-while loop?

âœ“ Correct Answer: Compilation error

The semicolon after a do-while loop is mandatory. Syntax: do { } while (condition); Forgetting it causes a compilation error. This is unique to do-while; other loops don't have a semicolon after the closing brace.

Q22. What is true about the for loop syntax: for(init; condition; update)?

âœ“ Correct Answer: All parts are optional

All three parts of a for loop are optional. for(;;) creates an infinite loop (equivalent to while(true)). You can initialize variables outside, check conditions inside the loop, and update manually. However, using all three parts is more readable.

Q23. What is a limitation of the enhanced for loop (for-each)?

âœ“ Correct Answer: You cannot modify the loop variable

In an enhanced for loop, the loop variable is a copy. Modifying it doesn't change the original array/collection element. Also, you don't have access to the index. For these cases, use a traditional for loop.

Q24. What feature allows switch to return a value directly?

âœ“ Correct Answer: switch expression (Java 12+)

Java 12+ introduced switch expressions that can return values directly using arrow syntax (->). Example: int day = switch(num) { case 1 -> 7; case 2 -> 6; default -> 0; }; This is more concise than traditional switch statements.

Q25. What is for(;;) equivalent to?

âœ“ Correct Answer: Infinite loop

for(;;) creates an infinite loop, equivalent to while(true). Since the condition is empty, it's treated as true. This is sometimes used intentionally with break statements inside for server loops or continuous processing.

Q26. What does return do inside a loop?

âœ“ Correct Answer: It exits the current method

The return statement exits the entire method, not just the loop. If you only want to exit the loop, use break. If you want to skip the current iteration, use continue. Return is used when you've found your answer and want to exit the method.

Q27. What is the most common update expression in a for loop?

âœ“ Correct Answer: i++

i++ (post-increment) is the most common update expression. While i++, ++i, and i = i + 1 all increment i, i++ is conventional. In a for loop update section, there's no practical difference between i++ and ++i.

Q28. What type must the condition in an if statement evaluate to?

âœ“ Correct Answer: boolean

The condition in an if statement must evaluate to a boolean (true or false). Unlike C/C++, you cannot use integers (0 or 1). This prevents common bugs where assignment (=) is mistaken for comparison (==).

Q29. What is the best practice when you have only two cases in a switch?

âœ“ Correct Answer: Use if-else for better readability

For only two cases, if-else is more readable than switch. Switch is best for 3+ cases. For simple value assignment with two cases, consider the ternary operator. Choose the construct that makes your intent clearest.

Q30. What is the best practice for nested loops?

âœ“ Correct Answer: Avoid deep nesting; extract to methods

Avoid deep nesting (more than 2-3 levels) as it reduces readability and increases complexity. Extract inner loops into separate methods with descriptive names. This makes code more maintainable and testable.
