Q1. What does JDBC stand for?

âœ“ Correct Answer: Java Database Connectivity

JDBC stands for Java Database Connectivity, which is a standard Java API for database-independent connectivity between the Java programming language and a wide range of databases.

Q2. Which package contains the core JDBC classes and interfaces?

âœ“ Correct Answer: java.sql

The `java.sql` package contains the core JDBC API, including `Connection`, `Statement`, `ResultSet`, and `DriverManager`.

Q3. Which class is responsible for managing a list of database drivers?

âœ“ Correct Answer: DriverManager

`DriverManager` is the basic service for managing a set of JDBC drivers. It attempts to locate a suitable driver from the loaded drivers.

Q4. Which type of JDBC driver is known as the "Thin Driver" and converts JDBC calls directly into the vendor-specific database protocol?

âœ“ Correct Answer: Type 4

Type 4 drivers (Native-Protocol/All-Java Driver) convert JDBC calls directly into the vendor-specific database protocol. They are written entirely in Java and are platform-independent.

Q5. Which interface represents a session with a specific database?

âœ“ Correct Answer: Connection

The `Connection` interface represents a session with a specific database. SQL statements are executed and results are returned within the context of a connection.

Q6. Which interface is used to execute precompiled SQL statements?

âœ“ Correct Answer: PreparedStatement

`PreparedStatement` is used to execute precompiled SQL statements. It is more efficient for repeated execution and prevents SQL injection.

Q7. Which interface represents the result set of a database query?

âœ“ Correct Answer: ResultSet

A `ResultSet` object maintains a cursor pointing to its current row of data. It provides methods to retrieve column values from the current row.

Q8. Which interface is used to execute stored procedures?

âœ“ Correct Answer: CallableStatement

`CallableStatement` is used to execute SQL stored procedures.

Q9. Which method is used to execute a SELECT statement?

âœ“ Correct Answer: executeQuery()

`executeQuery()` is used to execute statements that return a single `ResultSet` object, typically SELECT statements.

Q10. Which method is used to execute INSERT, UPDATE, or DELETE statements?

âœ“ Correct Answer: executeUpdate()

`executeUpdate()` is used for SQL statements that return a row count (INSERT, UPDATE, DELETE) or nothing (DDL statements).

Q11. Which method is commonly used to register a JDBC driver class explicitly?

âœ“ Correct Answer: Class.forName()

`Class.forName("com.mysql.cj.jdbc.Driver")` is the traditional way to load and register a JDBC driver. In modern JDBC (4.0+), this is often automatic via SPI.

Q12. How do you disable auto-commit mode to manage transactions manually?

âœ“ Correct Answer: setAutoCommit(false)

By default, new connections are in auto-commit mode. Calling `setAutoCommit(false)` allows you to group multiple statements into a single transaction.

Q13. Which method is used to save changes made in the current transaction?

âœ“ Correct Answer: commit()

`commit()` makes all changes made since the previous commit/rollback permanent and releases any database locks currently held by the Connection.

Q14. Which method is used to undo changes made in the current transaction?

âœ“ Correct Answer: rollback()

`rollback()` undoes all changes made in the current transaction and releases any database locks.

Q15. What feature allows executing a group of SQL statements together for performance?

âœ“ Correct Answer: Batch Processing

Batch processing allows you to group related SQL statements into a batch and submit them with a single call to the database, improving performance.

Q16. Which method is used to add a SQL command to the current batch?

âœ“ Correct Answer: addBatch()

`addBatch()` adds the given SQL command to the current list of commands for the Statement object.

Q17. Which method moves the cursor to the next row in a ResultSet?

âœ“ Correct Answer: next()

`next()` moves the cursor forward one row from its current position. It returns `true` if the new current row is valid; `false` if there are no more rows.

Q18. Which checked exception is thrown by most JDBC methods?

âœ“ Correct Answer: SQLException

`SQLException` is the general exception thrown when a database access error or other errors occur.

Q19. Which is the best defense against SQL Injection attacks in JDBC?

âœ“ Correct Answer: PreparedStatement

Using `PreparedStatement` with parameterized queries is the most effective way to prevent SQL injection, as the database treats parameters as data, not executable code.

Q20. Which interface is preferred over DriverManager for obtaining connections in enterprise applications?

âœ“ Correct Answer: DataSource

`DataSource` is the preferred way to get a connection. It can provide connection pooling and distributed transaction support.

Q21. What technique is used to reuse database connections to improve performance?

âœ“ Correct Answer: Connection Pooling

Connection pooling maintains a cache of database connections so that they can be reused when future requests to the database are required, avoiding the overhead of establishing a new connection.

Q22. Which ResultSet type allows the cursor to move both forward and backward?

âœ“ Correct Answer: TYPE_SCROLL_INSENSITIVE

`TYPE_SCROLL_INSENSITIVE` (and `TYPE_SCROLL_SENSITIVE`) allows the cursor to move forward and backward, and also to jump to a specific row.

Q23. Which concurrency mode allows updates to the ResultSet to be reflected in the database?

âœ“ Correct Answer: CONCUR_UPDATABLE

`CONCUR_UPDATABLE` allows changes made to the `ResultSet` object (using update methods) to be saved to the underlying database.

Q24. Which interface provides comprehensive information about the database as a whole?

âœ“ Correct Answer: DatabaseMetaData

`DatabaseMetaData` provides information about the database, such as table names, column names, supported features, and version information.

Q25. Which interface provides information about the types and properties of the columns in a ResultSet?

âœ“ Correct Answer: ResultSetMetaData

`ResultSetMetaData` can be used to get information about the types and properties of the columns in a `ResultSet` object.

Q26. What is the most important step to do in a finally block when using JDBC?

âœ“ Correct Answer: close()

It is crucial to close JDBC resources (`ResultSet`, `Statement`, `Connection`) in a `finally` block (or try-with-resources) to prevent resource leaks.

Q27. What is the Type 1 JDBC driver also known as?

âœ“ Correct Answer: JDBC-ODBC Bridge

The Type 1 driver translates JDBC calls into ODBC calls. It is generally considered obsolete and was removed in Java 8.

Q28. Which method can be used to give the JDBC driver a hint as to the number of rows that should be fetched from the database?

âœ“ Correct Answer: setFetchSize()

`setFetchSize()` gives the JDBC driver a hint as to the number of rows that should be fetched from the database when more rows are needed.

Q29. Which interface allows you to set a point within a transaction to which you can later roll back?

âœ“ Correct Answer: Savepoint

The `Savepoint` interface allows you to set a savepoint within a transaction, enabling partial rollbacks to that specific point.

Q30. Which method is used to retrieve auto-generated keys after an INSERT statement?

âœ“ Correct Answer: getGeneratedKeys()

`getGeneratedKeys()` returns a `ResultSet` containing the auto-generated keys created as a result of executing a `Statement`.
