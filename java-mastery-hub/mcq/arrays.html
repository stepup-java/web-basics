Once an array is created in Java, its size cannot be changed. If you need a resizable collection, use ArrayList.

Q2. Which is the correct way to declare and initialize an array of 5 integers?

âœ“ Correct Answer: int[] arr = new int[5];

The correct syntax uses square brackets [] for declaration and new type[size] for initialization.

Q3. What is the index of the first element in a Java array?

âœ“ Correct Answer: 0

Java arrays are 0-indexed, meaning the first element is at index 0.

Q4. How do you get the number of elements in an array named 'arr'?

âœ“ Correct Answer: arr.length

Arrays have a length property (not a method) that returns the size. Strings use length() method.

Q5. What is the default value of elements in an int array?

âœ“ Correct Answer: 0

When an array of integers is created, all elements are initialized to 0 by default.

Q6. What exception is thrown if you access arr[5] in an array of size 5?

âœ“ Correct Answer: ArrayIndexOutOfBoundsException

Valid indices for an array of size 5 are 0 to 4. Accessing index 5 throws ArrayIndexOutOfBoundsException.

Q7. What is the default value of elements in a String array?

âœ“ Correct Answer: null

String is an object type, so the default value for elements in a String array is null.

Q8. How do you declare a 2D array?

âœ“ Correct Answer: Both B and C

Both int[][] name and int name[][] are valid declarations in Java, though the first one is preferred.

Q9. What do you call an array where rows have different lengths?

âœ“ Correct Answer: Jagged Array

A jagged array is a multidimensional array where member arrays are of different sizes.

Q10. How do you print the contents of an array arr nicely?

âœ“ Correct Answer: Arrays.toString(arr)

System.out.println(arr) prints the memory address. Arrays.toString(arr) prints the elements.

Q11. What is the time complexity to access an element by index?

âœ“ Correct Answer: O(1)

Arrays use contiguous memory, so accessing an element by index is an instant (constant time) operation.

Q12. Which method sorts an array in ascending order?

âœ“ Correct Answer: Arrays.sort(arr)

java.util.Arrays.sort() is the standard method to sort arrays.

âœ“ Correct Answer: The array must be sorted

Binary search algorithm only works on sorted data. If the array is not sorted, the result is undefined.

Q14. Where are array objects stored in memory?

âœ“ Correct Answer: Heap

In Java, all objects (including arrays) are stored in the Heap memory. The reference to the array is stored in the Stack.

Q15. How can you create a larger copy of an existing array?

âœ“ Correct Answer: Arrays.copyOf(arr, newLength)

Arrays.copyOf() creates a new array of the specified length and copies elements from the original array.

Q16. What is the direct superclass of an array type?

âœ“ Correct Answer: Object

Every array type in Java is a direct subclass of java.lang.Object.

Q17. How do you check if two arrays contain the same elements?

âœ“ Correct Answer: Arrays.equals(a, b)

a == b checks reference equality. Arrays.equals(a, b) checks content equality.

Q18. Which method converts an array to a List?

âœ“ Correct Answer: Arrays.asList(arr)

Arrays.asList() returns a fixed-size list backed by the specified array.

Q19. How do you fill an entire array with the value 1?

Arrays.fill() is a utility method to assign a specified value to each element of an array.

Arrays in Java implement the Cloneable and java.io.Serializable interfaces.

For 1D arrays of primitives, it's a deep copy. For arrays of objects (including 2D arrays), it's a shallow copy (references are copied).

Arrays are covariant, meaning Integer[] is a subtype of Number[]. Generics are invariant.

Q29. How do you convert an array to a Stream?

âœ“ Correct Answer: Both B and C

Arrays.stream(arr) creates a stream from the array. Stream.of(arr) also works (but for primitives it creates a Stream of the array object itself, not elements).

Q30. Which is the most efficient way to copy a large array?

System.arraycopy() is a native method and is generally the fastest way to copy arrays.
