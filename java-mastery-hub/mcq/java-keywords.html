Q1. How many reserved keywords does Java have (as of Java 17)?

âœ“ Correct Answer: 50

Java has 50 reserved keywords. These are words that have special meaning in Java and cannot be used as identifiers (variable names, method names, class names, etc.). Keywords like const and goto are reserved but not used.

Q2. Which access modifier allows a class member to be accessible from anywhere?

âœ“ Correct Answer: public

The public access modifier makes a class member accessible from anywhere - from the same class, same package, subclasses, and even different packages. It provides the widest level of access.

Q3. Which access modifier provides the most restrictive access?

âœ“ Correct Answer: private

The private access modifier is the most restrictive. Private members are accessible only within the same class where they are declared. This is fundamental to encapsulation in OOP.

Q4. Where can protected members be accessed?

âœ“ Correct Answer: Same package and subclasses in any package

Protected members are accessible within the same package and by subclasses in any package (through inheritance). This is more permissive than default but more restrictive than public.

Q5. What is the scope of default (package-private) access modifier?

âœ“ Correct Answer: Only within the same package

When no access modifier is specified (default/package-private), the member is accessible only within the same package. It's not accessible from subclasses in different packages, unlike protected.

Q6. Which keyword is used to create a class-level variable that is shared among all instances?

âœ“ Correct Answer: static

The static keyword creates class-level members that belong to the class itself rather than to instances. Static variables are shared among all instances of the class and can be accessed without creating an object.

Q7. Which keyword prevents a variable from being reassigned after initialization?

âœ“ Correct Answer: final

The final keyword makes a variable constant - once assigned, it cannot be reassigned. For reference types, the reference cannot change, but the object's state can still be modified. Note: 'const' is a reserved keyword but not used in Java.

Q8. Which keyword is used to declare a class that cannot be instantiated?

âœ“ Correct Answer: abstract

The abstract keyword declares a class that cannot be instantiated directly. Abstract classes are meant to be extended by subclasses. They can contain both abstract methods (without implementation) and concrete methods (with implementation).

Q9. Which keyword refers to the current instance of a class?

âœ“ Correct Answer: this

The 'this' keyword refers to the current instance of the class. It's used to access instance variables and methods, differentiate between instance variables and parameters with the same name, and call other constructors in the same class.

Q10. Which keyword is used to call the parent class constructor?

âœ“ Correct Answer: super

The 'super' keyword is used to call the parent class constructor (super()) or access parent class members (super.methodName()). When calling the parent constructor, super() must be the first statement in the child constructor.

Q11. Which keyword is used to create an object instance?

âœ“ Correct Answer: new

The 'new' keyword is used to create new objects (instances) of a class. It allocates memory on the heap for the object and calls the constructor. Example: Student s = new Student();

Q12. Which keyword is used to ensure thread-safe access to a method or block?

âœ“ Correct Answer: synchronized

The synchronized keyword ensures that only one thread can execute a method or block at a time. It provides mutual exclusion and prevents race conditions in multi-threaded applications.

Q13. Which keyword ensures that a variable's value is always read from main memory?

âœ“ Correct Answer: volatile

The volatile keyword ensures that a variable's value is always read from and written to main memory, not from thread-local cache. This guarantees visibility of changes across threads but doesn't provide atomicity.

Q14. Which keyword prevents a field from being serialized?

âœ“ Correct Answer: transient

The transient keyword marks a field to be excluded from serialization. When an object is serialized, transient fields are not saved. This is useful for sensitive data or fields that can be recalculated.

Q15. Which keyword is used to test if an object is an instance of a specific class?

âœ“ Correct Answer: instanceof

The instanceof keyword tests whether an object is an instance of a specific class or implements a specific interface. It returns true if the object is an instance, false otherwise. Example: if (obj instanceof String)

Q16. Which keyword is used to inherit from a class?

âœ“ Correct Answer: extends

The extends keyword is used for class inheritance. A class can extend only one parent class (single inheritance). Example: class Dog extends Animal. The child class inherits all non-private members of the parent class.

Q17. Which keyword is used to implement an interface?

âœ“ Correct Answer: implements

The implements keyword is used when a class implements an interface. A class can implement multiple interfaces (separated by commas), providing implementations for all abstract methods. Example: class MyClass implements Interface1, Interface2

Q18. Which keyword is used to exit from a method and optionally return a value?

âœ“ Correct Answer: return

The return keyword exits from a method and optionally returns a value to the caller. For void methods, use 'return;' without a value. For non-void methods, you must return a value of the declared return type.

Q19. Which keyword is used to exit from a loop or switch statement?

âœ“ Correct Answer: break

The break keyword immediately exits from a loop (for, while, do-while) or switch statement. Control transfers to the statement immediately following the loop or switch. It can also be used with labels to break out of nested loops.

Q20. Which keyword skips the current iteration of a loop and continues with the next?

âœ“ Correct Answer: continue

The continue keyword skips the remaining code in the current iteration and moves to the next iteration of the loop. It's useful when you want to skip certain iterations based on a condition without exiting the entire loop.

Q21. Which keywords are used for exception handling?

âœ“ Correct Answer: try, catch, finally

Exception handling in Java uses try (to wrap risky code), catch (to handle exceptions), and finally (to execute cleanup code regardless of whether an exception occurred). The finally block is optional but always executes.

Q22. Which keyword is used to explicitly throw an exception?

âœ“ Correct Answer: throw

The 'throw' keyword is used to explicitly throw an exception. Example: throw new IllegalArgumentException("Invalid input"); It's used inside method bodies to throw exceptions when error conditions are detected.

Q23. Which keyword is used in a method signature to declare that it may throw exceptions?

âœ“ Correct Answer: throws

The 'throws' keyword is used in method signatures to declare that the method may throw certain checked exceptions. Example: public void readFile() throws IOException. Callers must either handle these exceptions or declare them in their own throws clause.

Q24. What does the final keyword do when applied to a method?

âœ“ Correct Answer: It prevents method overriding

When applied to a method, the final keyword prevents subclasses from overriding that method. This is useful when you want to ensure that the method's implementation remains unchanged in all subclasses.

Q25. What does the final keyword do when applied to a class?

âœ“ Correct Answer: It prevents inheritance

When applied to a class, the final keyword prevents other classes from extending it. The class cannot be subclassed. Examples of final classes in Java include String, Integer, and other wrapper classes.

Q26. Which keyword ensures consistent floating-point calculations across platforms?

âœ“ Correct Answer: strictfp

The strictfp (strict floating-point) keyword ensures that floating-point calculations produce identical results on all platforms. It can be applied to classes, interfaces, or methods. This is rarely used in modern Java.

Q27. Which keyword indicates that a method is implemented in platform-specific code (C/C++)?

âœ“ Correct Answer: native

The native keyword indicates that a method is implemented in platform-specific code (usually C or C++) using JNI (Java Native Interface). Native methods have no body in Java and are declared with a semicolon. Example: public native void nativeMethod();

Q28. Which keyword is used for debugging to test assumptions in code?

âœ“ Correct Answer: assert

The assert keyword is used to test assumptions during development. If the assertion fails, an AssertionError is thrown. Assertions can be enabled/disabled at runtime with -ea/-da flags. Example: assert x > 0 : "x must be positive";

Q29. Are 'goto' and 'const' valid keywords in Java?

âœ“ Correct Answer: No, they are reserved but not used

'goto' and 'const' are reserved keywords in Java but are not used. They cannot be used as identifiers. Java reserves them to prevent confusion for programmers coming from C/C++. Use 'final' instead of 'const' for constants.

Q30. Which keyword is used to declare a package in Java?

âœ“ Correct Answer: package

The package keyword is used to declare a package at the beginning of a Java file. Packages organize classes into namespaces and provide access protection. Example: package com.example.myapp; The package statement must be the first statement in the file.
