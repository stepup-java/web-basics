Q3. Write a program to find the second largest number in an array.


                    // Sample Input:
                    // 10 50 30 90 20
                    // Sample Output:
                    // Second Largest: 50
ðŸ’¡ Show Answer 

int[] arr = {10, 50, 30, 90, 20};
int max = Integer.MIN_VALUE, secondMax = Integer.MIN_VALUE;
for (int num : arr) {
    if (num > max) {
        secondMax = max;
        max = num;
    } else if (num > secondMax && num != max) {
        secondMax = num;
    }
}
System.out.println("Second Largest: " + secondMax);

Explanation: Maintains two variables for largest and second largest.

Q6. Write a program to reverse an array.


                    // Sample Input:
                    // 1 2 3 4 5
                    // Sample Output:
                    // [5, 4, 3, 2, 1]
ðŸ’¡ Show Answer 

int[] arr = {1, 2, 3, 4, 5};
int start = 0, end = arr.length - 1;
while (start < end) {
    int temp = arr[start];
    arr[start] = arr[end];
    arr[end] = temp;
    start++;
    end--;
}
System.out.println(java.util.Arrays.toString(arr));

Explanation: Swaps elements from both ends moving towards center.

Q11. Write a program to count vowels and consonants in a String.


                    // Sample Input:
                    // Hello World
                    // Sample Output:
                    // Vowels: 3, Consonants: 7
ðŸ’¡ Show Answer

str = str.toLowerCase();
for (int i = 0; i < str.length(); i++) {
    char ch = str.charAt(i);
    if (ch >= 'a' && ch <= 'z') {
        if (ch == 'a' || ch == 'e' || ch == 'i' || ch == 'o' || ch == 'u') v++;

    }

Explanation: Iterates through string checking characters.
