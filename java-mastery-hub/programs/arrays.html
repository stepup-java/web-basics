Program 1: Basic Declaration & Initialization

Different ways to create an array.

â˜• Java
public class ArrayBasics {
    public static void main(String[] args) {
        // Method 1: Declaration then Initialization
        int[] numbers;
        numbers = new int[3];
        numbers[0] = 10;
        numbers[1] = 20;
        numbers[2] = 30;

}

Technical: new int[3] allocates memory for 3 integers on the heap. {...} is an array initializer shortcut.

Program 3: Iterating (For Loop & Enhanced For Loop)

Looping through array elements.

â˜• Java
public class ArrayIteration {
    public static void main(String[] args) {
        String[] colors = {"Red", "Green", "Blue"};

        // Traditional For Loop
        System.out.println("Using For Loop:");
        for (int i = 0; i < colors.length; i++) {
            System.out.println("Color at index " + i + ": " + colors[i]);

        }
    }
}

Baby: Looking at each color card one by one.

Technical: Enhanced for-loop is cleaner but doesn't give access to the index.

Program 4: Finding Sum and Average

Calculating aggregate values.

public class SumAverage {

Program 5: Finding Max and Min

Finding the largest and smallest numbers.

public class MaxMin {
    public static void main(String[] args) {
        int[] numbers = {15, 8, 23, 42, 4};
        
        int max = numbers[0];
        int min = numbers[0];

        for (int i = 1; i < numbers.length; i++) {

}

Baby: Finding the tallest and shortest kid in line.

Program 6: Reversing an Array

Flipping the order of elements.

â˜• Java
import java.util.Arrays;

public class ReverseArray {
    public static void main(String[] args) {
        int[] arr = {1, 2, 3, 4, 5};
        
        // Two-pointer technique
        int start = 0;
        int end = arr.length - 1;
        
        while (start < end) {
            // Swap
            int temp = arr[start];
            arr[start] = arr[end];
            arr[end] = temp;
            
            start++;
            end--;
        }
        
        System.out.println("Reversed: " + Arrays.toString(arr));
    }
}

Baby: Swapping the first and last person in line until everyone is flipped.

Real-world: Reversing the order of a playlist.

Technical: In-place reversal using two pointers avoids creating a new array (O(1) space).

Program 7: 2D Array (Matrix)

Working with rows and columns.

â˜• Java
public class MatrixDemo {
    public static void main(String[] args) {
        int[][] matrix = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };

        // Nested loops to print matrix
        for (int i = 0; i < matrix.length; i++) { // Rows
            for (int j = 0; j < matrix[i].length; j++) { // Columns
                System.out.print(matrix[i][j] + " ");
            }
            System.out.println(); // New line after each row
        }
    }
}

Baby: A grid like a tic-tac-toe board.

Real-world: A spreadsheet or a chessboard.

Technical: A 2D array is an array of arrays. matrix[row][col] accesses elements.

Program 10: Arrays.binarySearch()

Fast searching in a sorted array.

import java.util.Arrays;

public class SearchDemo {
    public static void main(String[] args) {

Program 15: Student Grade Analyzer

A complete example combining concepts.

â˜• Java
import java.util.Arrays;

public class GradeAnalyzer {
    public static void main(String[] args) {
        int[] grades = {85, 92, 78, 64, 95, 88};
        
        // 1. Sort grades
        Arrays.sort(grades);
        System.out.println("Sorted Grades: " + Arrays.toString(grades));
        
        // 2. Find lowest and highest
        System.out.println("Lowest: " + grades[0]);
        System.out.println("Highest: " + grades[grades.length - 1]);
        
        // 3. Calculate average
        double sum = 0;
        for (int grade : grades) {
            sum += grade;
        }
        System.out.println("Average: " + (sum / grades.length));
        
        // 4. Check if anyone got 100
        int perfectScoreIndex = Arrays.binarySearch(grades, 100);
        if (perfectScoreIndex >= 0) {
            System.out.println("Someone got a perfect score!");
        } else {
            System.out.println("No perfect scores.");
        }
    }
}

Baby: A teacher organizing test papers to find the best student and the class average.

Real-world: A grading system for a class.

Technical: Combines sorting, access, iteration, and searching to perform data analysis.
